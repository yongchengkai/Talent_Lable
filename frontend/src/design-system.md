# Talent Label 前端设计规范 — AI-Native Cyberpunk

## 设计理念

融合 AI 时代的视觉语言，参考 Linear、Vercel、Raycast、Arc Browser 的设计精髓，
打造一套「深色科技感 + 流光渐变 + 微动效」的企业级 AI 原生界面。

## 色彩体系

### 背景层次
- 主背景：深空黑 `#08080f` → `#0d0d1a` 渐变
- 侧边栏：`#0a0a16`，带微弱噪点纹理
- 内容卡片：`rgba(255,255,255,0.03)` 半透明毛玻璃
- 悬浮面板：`rgba(255,255,255,0.05)` + backdrop-blur

### 主色调
- 主色渐变：Cyan `#06b6d4` → Blue `#3b82f6` → Violet `#8b5cf6`
- 强调色：Electric Cyan `#22d3ee`
- 辅助渐变：Pink `#ec4899` → Orange `#f97316`（用于警告/热力）

### 文字色
- 主文字：`rgba(255,255,255,0.92)`
- 次文字：`rgba(255,255,255,0.55)`
- 弱文字：`rgba(255,255,255,0.3)`
- 链接/操作：`#22d3ee`

### 状态色
- 成功：`#10b981`（翡翠绿）
- 警告：`#f59e0b`（琥珀）
- 错误：`#ef4444`（珊瑚红）
- 信息：`#06b6d4`（青色）

### 边框与分割
- 边框：`rgba(255,255,255,0.06)`
- 悬浮边框：`rgba(34,211,238,0.2)`
- 发光边框：`0 0 0 1px rgba(34,211,238,0.15), 0 0 20px rgba(34,211,238,0.05)`

## 动效规范

### 页面进入
- 整页淡入上滑：`opacity 0→1, translateY 16px→0`，duration 500ms，ease-out

### 表格行
- 交错入场：每行延迟 30ms，从左侧滑入 `translateX(-12px→0)`
- 悬浮效果：行背景渐变发光 + 左侧出现 3px 渐变指示条
- 选中行：持续发光边框 + 背景加深

### 按钮
- 主按钮：渐变背景 + hover 时上浮 2px + 发光阴影扩散
- 次按钮：透明背景 + 边框发光
- 点击：scale(0.97) 回弹

### 弹窗
- 背景：深色毛玻璃 `backdrop-filter: blur(20px)`
- 进入：scale(0.9→1) + opacity，spring 弹性曲线
- 卡片内容：微弱的渐变光晕装饰

### 侧边栏
- 选中项：左侧渐变指示条 + 背景渐变发光
- 顶部 Logo：持续微弱呼吸光效
- 分隔线：渐变发光线

## 组件规范

### 卡片
- 背景：半透明 + 1px 发光边框
- 悬浮：边框变亮 + 微弱上浮 + 阴影扩散
- 圆角：16px

### 输入框
- 背景：`rgba(255,255,255,0.04)`
- 边框：`rgba(255,255,255,0.08)`
- 聚焦：边框变为主色 + 外发光 + 背景微亮

### 标签 Tag
- 胶囊形：圆角 20px
- 背景：对应色的 10% 透明度
- 文字：对应色的 90% 亮度

### 表格
- 表头：半透明深色 + 大写字母间距
- 行：透明背景，悬浮时渐变发光
- 边框：极淡分割线
